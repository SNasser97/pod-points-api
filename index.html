About

Pod points is a simple fullstack web project developed by me using the Listen Notes API (of course with the help of google and documentation!) built throughout whilst being in qurantine.

This is my own personal project which I decided to work on and if any suggestions or improvements, feel free to fork and build on top of it

Do note you will need an API key from Listen Notes

Techstack:

  • React + Redux + Router
  • Node + Express
  • postgreSQL
  • SASS

Vendors

  • Listen Notes
  • React-media-player
  • react-reveal
